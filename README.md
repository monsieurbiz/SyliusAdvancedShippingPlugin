## Compatibility

| Sylius Version | PHP Version     |
|----------------|-----------------|
| 1.12           | 8.1 - 8.2 - 8.3 |
| 1.13           | 8.1 - 8.2 - 8.3 |
| 1.14           | 8.1 - 8.2 - 8.3 |

## Installation

Install the plugin with composer:

```
composer require monsieurbiz/sylius-advanced-shipping-plugin
```

And follow the [installation documentation](docs/installation.md).
